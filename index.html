2. Para Professores dos Cursos

Selecionando o perfil pelo menu será exibido os dados pessoais.


  • É possivel a alteração dos campos: NOME, CPF, TITULAÇÃO, LATTES, E-MAIL, TELEFONE, CATEGORIA FUNCIONAL.
  • NÃO é possivel a alteração dos campos: CAMPUS, CENTRO, COLEGIADO, ESTADO DE CONTA. Em caso de erro comunicar a administração.
  • Alteração de senha: Digitar senha nova, e salvar.
  • Limpar: Limpa todas das alteração que NÃO foram salvas.
  • Enviar: salva as alterações feitas.
